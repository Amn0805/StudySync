
# 🌐 StudySync – Modern Study Website

## 📘 Overview
**StudySync** is a clean and modern responsive website built using **HTML** and **CSS**.  
The project aims to provide a simple yet elegant landing page for a study-related platform that helps students connect, learn, and stay productive.

---

## ✨ Features
- 📱 Fully responsive design (works on all devices)
- 🎨 Modern UI with smooth layout
- 🧭 Functional navigation bar
- 💬 About and Contact sections
- ⚡ Lightweight and fast-loading
- 🔗 Social media footer links

---

## 🛠️ Technologies Used
- **HTML5** – Structure and content  
- **CSS3** – Styling and layout  
- **Google Fonts** – Custom typography  
- **Font Awesome** – Icons (optional)
